# Experiment 7

## Aim

Write an HTML code to demonstrate the usage of internal CSS.

## Code

### index.html

```html
<!DOCTYPE html>
<html>
	<head>
		<title>Internal CSS</title>
		<style>
			p { text-align:center;
				color:blue;
				font-size:50px;
				font-weight:bold;}
		</style>
	</head>
	<body>
<p>I'm implemented with Internal CSS</p>
	</body>
</html>
```
